A Python library implementing a dict-like object that writes to a JSON file as it is modified

***************
Installation
***************
::

    pip install deadjson
    pip3 install deadjson

***************
Examples
***************
Initialize

>>> import deadjson
>>> data = deadjson.Util(data="data",fp=True)
>>> print(data)
